Column name  (Description)
======================================
OBJECTID = OBJECTID  (Unique Identifier)
F_TYPE   = FEATURE_TYPE (Can be one of the following values "COMMON","CONDO","CORRIDOR","RESERVE") 




F_TYPE FEATURE DEFINITION
======================================

Condo is a closed boundary that represents the extent of ownership by a Condominium Corporation.

Corridor describes a section of Right of Way.

Reserve is a unit of land of varying width, but usually 0.3 metres (1 foot) wide, used to restrict access to private/public properties, such as access to adjoining public highways. 

Common is a municipal parcel that is not designated as Corridor, Reserve or Condominium. A subdivision lot is a typical Common parcel.
